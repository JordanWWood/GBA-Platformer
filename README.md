# GBA Platformer
  
I'm not the most experienced with C, let alone with the GBA. This was made using DevKitArm for the gameboy advanced.
  
CONTROLS:  
 z                     - Jump  
 Enter                 - Pause  
 Left/Right Arrow Keys - Move  
 Backspace             - Reset  
  
Objective:  
 Reach the top of the platfroms and touch the end point.
